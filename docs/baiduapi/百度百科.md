百度百科
更新时间：2025-12-05
组件描述
百度百科组件可以通过输入的词条内容或词条id查询到详细的相关的百科的内容。

使用方式：组件、API

计费方式：详情可见组件计费。

API使用
接口定义
URL
https://appbuilder.baidu.com/v2/baike/lemma/get_content
Method	GET
Content-Type	application/json
Authorization	请求签名(Bearer <AppBuilder API Key>)
X-Appbuilder-Request-Id	非必填。uuid。填写后方便追踪问题。
请求参数定义
字段
类型
是否必填
说明
search_type	str	是	检索类型。
可选值：
lemmaTitle: 按百科词条名
lemmaId: 按百科词条ID
search_key	str	是	检索关键字。与searchType对应的检索条件
请求示例
Plain Text复制
curl -XGET \
  --location 'https://appbuilder.baidu.com/v2/baike/lemma/get_content?search_type=lemmaTitle&search_key=刘德华' \
  --header 'Authorization: xxxxxx' \
  --header 'Content-Type: application/json'
  --header 'X-Appbuilder-Request-Id: xxxxxx'
响应体定义
异常响应会有响应码code

字段
类型
是否必填
说明
request_id	str	是	请求ID
code	str	是	0 表示成功，其他都是异常
message	str	否	错误消息
result		否	义项内容
正常响应，返回词条内容体如下

LemmaContentByIdResponseData 定义
字段
类型
是否必填
说明
lemma_id	int	是	词条ID
lemma_title	str	是	词条名
lemma_desc	str	是	义项描述
url	str	是	词条页URL；移动设备支持自动适配
summary	str	是	词条摘要（优先提取概述，无概述时提取正文前n个字，不超过400字节）
abstract_plain	str	是	词条概述（纯文本格式，\n换行）
abstract_html	str	是	词条概述（HTML格式）
abstract_structured	str	是	词条概述（结构化数据）
pic_url	str	是	概述图片URL
square_pic_url	str	是	概述图片URL（自动裁切方图，可用于分享、icon等）
square_pic_url_wap	str	是	移动端概述图片URL（自动裁切方图，可用于分享、icon等）
card	list[dict]	是	基本信息栏项目
card_type	int	是	基本信息栏类型
albums	list[Album]	是	图册列表
classify	list[string]	是	词条分类，如：["演员","体育人物"]
star_map	list[StarMap]	是	星图列表
videos	List	是	视频列表
relations	List	是	关系列表
LemmaContentVideoData 定义
字段
类型
是否必填
说明
second_id	int	是	秒懂ID
cover_pic_url	str	是	封面原图
second_title	str	是	秒懂词条名
page_url	str	是	视频播放页URL
LemmaRelationData 定义
字段
类型
是否必填
说明
lemma_title	str	是	客体词条名
lemma_id	int	是	客体词条ID
relation_name	str	是	关系名
square_pic_url	str	是	客体词条方图
