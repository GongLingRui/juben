百科词条
更新时间：2025-12-01
组件描述
百度词条组件可以通过输入的词条内容查询到相关的百科词条列表。

使用方式：组件、API

计费方式：详情可见组件计费

API使用
接口定义
URL
https://appbuilder.baidu.com/v2/baike/lemma/get_list_by_title
Method	GET
Content-Type	application/json
Authorization	请求签名(Bearer <AppBuilder API Key>)
X-Appbuilder-Request-Id	非必填。uuid。填写后方便追踪问题。
请求参数定义
字段
类型
是否必填
说明
lemma_title	str	是	词条名
top_k	int	否	返回的结果数量。默认值：5。取值范围：[1，100]
请求示例
Plain Text复制
curl -XGET \
  --location 'https://appbuilder.baidu.com/v2/baike/lemma/get_list_by_title?lemma_title=刘德华' \
  --header 'Authorization: xxxxxx' \
  --header 'Content-Type: application/json'
  --header 'X-Appbuilder-Request-Id: xxxxxx'
响应体定义
字段
类型
是否必填
说明
request_id	str	是	请求ID
code	str	否	0 表示成功，其他都是异常
message	str	否	错误消息
result	object	否	义项列表
LemmaListResponseData 定义
字段
类型
是否必填
说明
lemma_id	int	是	词条ID
lemma_title	str	是	词条名
lemma_desc	str	是	义项描述
is_default	int	是	是否主义项，0-非主义项 1-主义项
url	str	是	词条页URL
