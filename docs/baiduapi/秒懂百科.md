秒懂百科
更新时间：2025-12-01
组件描述
秒懂百科：组件可以通过输入的词条内容或词条id查询到相关的视频内容。

使用方式：组件、API

计费方式：详情可见组件计费

API使用
接口定义
URL
https://appbuilder.baidu.com/v2/baike/second_know/search_video
Method	GET
Content-Type	application/json
Authorization	请求签名(Bearer <AppBuilder API Key>)
X-Appbuilder-Request-Id	非必填。uuid。填写后方便追踪问题。
请求参数定义
字段
类型
是否必填
说明
search_type	str	是	检索类型。
可选值：
lemmaTitle：按百科词条名
lemmaId ：按百科词条ID
search_key	str	是	检索关键字。与searchType对应的检索条件
limit	int	否	限制获取数量。默认值：1。
匹配到多个百科词条时，此参数指每个词条的限制数量。
type	int	否	视频类型。默认值：0，可选值：0 - 全部视频；1- 概述型视频
platform	str	否	视频来源。可以取多值，用逗号隔开，默认值：user ，可选值：user - 高级用户（默认）；app - App用户；partner - 合作方；pgc - PGC
请求示例
Plain Text复制
curl -XGET \
  --location 'https://appbuilder.baidu.com/v2/baike/second_know/search_video?search_type=lemmaTitle&search_key=刘德华' \
  --header 'Authorization: xxxxxx' \
  --header 'Content-Type: application/json'
  --header 'X-Appbuilder-Request-Id: xxxxxx'
响应体定义
错误响应会响应code 和message

字段
类型
是否必填
说明
request_id	str	是	请求ID
code	str	是	0 表示成功，其他都是异常
message	str	否	错误消息
result	object	否	返回的数据内容
正常响应，只响应秒懂视频数据列表

SecondKnowVideoSearchResponseData 定义
字段
类型
是否必填
说明
lemma_id	int	是	词条ID
lemma_title	str	是	词条名
lemma_desc	str	是	百科义项名
second_id	int	是	秒懂视频ID
second_title	str	是	秒懂视频标题
cover_pic_url	str	是	横版封面图URL
forever_play_url_mp4	str	是	视频永久播放URL（mp4）
play_time	int	是	时长（秒）
second_type	int	是	视频类型,（1：概述型，2：解释型，3：延展阅读；根据话题查询无此字段）
status	int	是	视频状态（0，新建转码中，1:转码完成，2:删除，1001:上线，1002:下线）
is_vertical	int	是	横竖屏属性, 0 横屏，1竖屏
cover_pic	dict	是	封面图数据，只有一个元素src，为hiphotos的加密ID
play_url_mp3	int	是	音频播放URL（mp3），时效URL，一定时间后会过期
play_url_mp4	int	是	视频播放URL（mp4），时效URL，一定时间后会过期
